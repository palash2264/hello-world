Grid Layout

For ease of development the following SASS includes are provided that contain defined columns.This @include grid__column(6); specifies the number of columns an element should span in the 12 column grid. Make sure the column include is nested with its parent element passing @include grid__container; as well

For more examples of how to extend upon this please refer to the bourbon/Neat documentation

Scss

Below you will find examples of the grid patterns to compose a group of elements into commonly seen layouts


          .main__custom.container {
          @include grid__container;
          }
          .layout__two-column--name {
          @include grid__column(6);
          }
