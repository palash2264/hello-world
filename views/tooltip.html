Web Design Styleguide

Tooltip

Styleguide tooltips take the form of a mixin that take a border color and whether the tip should be on the top or bottom (can be extended later). It is up to the each team to design the user experience for the tooltip display, for example on hover, click, etc.

Tooltip Top 

.tooltip-top { @include tooltip("interactive-blue", "top"); }
Tooltip Top With Text

Tooltip Bottom 

.tooltip-bottom { @include tooltip("interactive-blue", "bottom"); }
Tooltip Bottom With Text

Tooltip No Arrow 

.tooltip-no-arrow { @include tooltip("interactive-blue"); }
