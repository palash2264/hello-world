Web Design Styleguide

Notifications

Placeholder classes are available in _notifications.scss for the base notification styles available for Styleguide.

Errors

To create static CSS classes for error notifications you can defined them in your Sass by extending the placeholder styles.


.notification {
  &.-hard-interruption {
    @extend %hard-interruption;
  }

  &.-soft-interruption {
    @extend %soft-interruption;
  }

  &.-system-error {
    @extend %system-error;
  }

  &.-validation-error {
    @extend %validation-error;
  }
}

These classes will be processed and can be used in this fashion. When an element has a

role=alert is displayed it, the browser will send out an accessible alert event to assistive technology products which can then notify the user about it. Use role=alert for a one-time error notification which shows for a period of time and goes away and is intended to alert the user that something has happened.

Use this for input/validation input errors.

  • Missing phone number. Link to phone number form field.
  • Please provide a valid zip code. Link to zip code form field.
