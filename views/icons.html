Styled Usage with Mixins

We've implemented a variety of mixins to be used for styling icons with a circle or border. Below are some examples of how they would be used in your stylesheets and the flexibility the mixin will give you in choosing a CSS class name.

Feature Icons

The feature-icon mixin results in a 30px icon surrounded by a 60px circle on desktop screens and a 25px icon surrounded by a 50px circle on mobile devices. You can specify the color of the circle's background as a string parameter representing one of the colors defined in Styleguide's color map.

.appointment-feed-icon { @include feature-icon("moss"); }
prescription

Hollow Icons

Icons surrounded by a colored border can be generated by using the same mixin and passing the border color as the first parameter and "hollow" style as a second parameter.
Desktop icons are 30px and surrounded by a 3px border. Mobile devices display a 25px icon surrounded by a 2px border. 

.sidebar-icon { @include feature-icon("graphite-dark", "hollow");  }
stethoscope

Compact Icons

The feature icon mixin accepts the "compact" as a style parameter to generate compact icons.
Compact icons are always 20px surrounded by a 40px circle regardless of device.

.copay-icon { @include feature-icon("interactive-blue", "compact"); }
