Form Controls

Always associate a label with a form field with the purpose of the field.

Base styling for checkboxes can be applied with the form-control class. These checkboxes are 20px squares.



To create checkboxes of a different size, include the following mixin in your Sass. You must provide a classname, box size, and optionally a font-size.


          .my-30-px-checkbox {
          @include checkbox(30px, $medium-font-size);
          }

Append the class name "radio-button" to apend the styles of to your project radio buttons. Use the attribute checked="checked" to preselect a radio button. Basic radio buttons that make sense from their labels alone and use matching name attribute, do not require fieldset or legend.


Fieldset and legendshould only be used to associate groups of controls when a higher level description is needed. Do not use nested fieldsets.
