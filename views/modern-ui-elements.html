Accordion

Accordions are a treated as a group of expandable items in which opening one will reveal content associated with its label. The markup below is required to acheive the functionality written in lib/scripts/modern/accordion.js

The expected keyboard functionality is required in order to make this pattern keyboard accessible:

  • TAB key to enter/exit accordion
  • ARROW keys to change tabs
  • and ENTER or SPACEBAR to expand/collapse tabs
import accordion from 'styleguide/lib/modern/scripts/accordion';
accordion($("#hipstaccordion"));

Dropdown

Dropdown that contains a list of links used for selection. Please find the corresponding Javascript below. 


            $('.select-dropdown').each(function() {
              var $element = $(this);
              var $select = $element.find('select');
              var $value = $element.find('.select-value');
              // Bind event handlers to select
              $select.on({
                'change keyup': function() {
                  $value.text($select.val());
                },
                'focus': function() {
                  $element.addClass('focus');
                },
                'blur': function() {
                  $element.removeClass('focus');
                }
              });
              $select.trigger('change');
            });

Add .-form to the dropdown to style it as the main dropdown used on the top of many features, meant to be used in forms.
